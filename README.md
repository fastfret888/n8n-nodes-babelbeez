# n8n-nodes-babelbeez

n8n community nodes for Babelbeez voice agent automation.

> This repository is public. Do not commit `.env` files, real API keys, webhook signing secrets, n8n credential exports, or customer data.

## Development

This project uses nvm for repo-local Node.js isolation.

```bash
nvm install
nvm use
```

The n8n node implementation will be scaffolded with the official `@n8n/node` tooling.
